Storage Features:
  ✅ Shadow Clones for VDI boot storm optimization
  ✅ Inline compression uses LZ4 (delay=0)
  ✅ Post-process compression uses delay > 0
  ✅ Erasure Coding best for cold/read-heavy data

3. ❌ Mixing Stargate (I/O) with Curator (GC)
   ✅ Stargate = real-time storage I/O
   ✅ Curator = background maintenance tasks

4. ❌ Using physical port instead of logical network names
   ✅ Use acli net.list (not ovs-vsctl alone)

5. ❌ Misunderstanding compression delays
   ✅ delay=0 means inline (immediate, LZ4)

7. ❌ Missing service port numbers in troubleshooting
