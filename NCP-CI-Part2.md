### Q1
An administrator is deploying NC2 on AWS and needs to allow browser-based management access to Prism Element from a corporate jump host. Which inbound security group rule must be added to the NC2 cluster's security group?

- A) Allow TCP port 9440 from the jump host's CIDR
- B) Allow TCP port 443 from 0.0.0.0/0
- C) Allow TCP port 8443 from the jump host's CIDR
- D) Allow TCP port 2049 from the VPC CIDR

**Answer: A**
Prism Element and Prism Central use TCP port 9440 for HTTPS management access. Port 443 is not the Prism default, port 8443 is not used by Prism, and port 2049 is NFS.

### Q2
A team is running NC2 on AWS and needs their CVM-hosted volumes to be accessible from VMs in a peered application VPC. Which port must be allowed inbound on the NC2 security group?

- A) TCP 3260 for iSCSI target access
- B) TCP 2049 for NFS export access
- C) UDP 3261 for iSCSI discovery
- D) TCP 445 for SMB share access

**Answer: A**
Nutanix Volumes (ABS) uses the iSCSI protocol on TCP port 3260. Allowing this port in the security group enables VMs in peered VPCs to connect to Nutanix-hosted iSCSI targets.

### Q3

AWS VPC peering requires non-overlapping CIDR ranges between the two VPCs. Since both VPCs use 10.10.0.0/16, the peering connection cannot route traffic correctly.

### Q4

- A) AWS Transit Gateway
- B) AWS PrivateLink
- C) AWS Global Accelerator
- D) AWS CloudFront

AWS Transit Gateway acts as a central hub that connects multiple VPCs in a hub-and-spoke topology. This eliminates the need for full-mesh VPC peering and simplifies routing for multi-VPC architectures.
